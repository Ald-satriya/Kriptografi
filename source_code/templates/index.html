System Overview

Sistem Tanda Tangan Digital PDF

Sistem ini mengimplementasikan algoritma kriptografi RSA dengan fungsi hash SHA-256 untuk membuat dan memverifikasi tanda tangan digital pada dokumen PDF.

Fitur Utama:
  • Tanpa Login: Sistem langsung dapat digunakan
  • RSA 2048-bit: Algoritma kriptografi asimetris
  • SHA-256: Fungsi hash untuk integritas data
  • Web Interface: Antarmuka web yang responsif
  • Export Lengkap: Semua hasil dapat diexport untuk UAS
